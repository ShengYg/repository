### 4. lambda

~~~java
Collections.sort(list, (a, b) -> {
    if(a.left == b.left)
        return a.right - b.right;
    else
        return a.left - b.left;
});
~~~

### 5. map, reduce, filter

List<Integer> map_num = list.stream().map(num -> num * 2).collect(Collectors.toList());
